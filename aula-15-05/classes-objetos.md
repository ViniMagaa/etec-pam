### Getters e Setters
- **Getters:** Faz a leitura da instância
- **Setters:** Atualiza o valor da instância

Sintaxe `get`:
```dart
datatype get get[Variável] {
    // Código
}
```

Sintaxe `set`:
```dart
set set [Variável] (argumento) {
    // Código
}
```
